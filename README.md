# YourOS

個人向け開発・業務OS。Claude Code中心の「思いつき→実行→検証→記憶」サイクルを回す。

## フォルダ構成

| フォルダ | 用途 |
|---------|------|
| Projects/ | 進行中プロジェクト（Spec/Tasks/Notes/Context） |
| Inbox/ | すべての入口。思いつきを即記録 |
| Knowledge/ | 短く要約された"正"（dev/biz/tools） |
| SOP/ | 再現可能な手順（dev/ops/security） |
| Decisions/ | 意思決定の台帳（横断参照） |
| Prompts/ | プロンプト雛形・スキル設計・定型指示 |
| Scratch/ | 作業中の一時置き場（週次で整理） |
| Archive/ | 完了・陳腐化した退避先 |
| Private/ | 個人情報・機密（AIアクセス不可） |
| AI-readable/ | AI向け整形済み正本・要約・索引 |
| AI-blocked/ | AIアクセス禁止領域（鍵・契約・顧客データ） |

詳細なAIルールは [CLAUDE.md](CLAUDE.md) を参照。
